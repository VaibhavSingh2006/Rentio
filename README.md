# 🏠 Rentio

✨ Rentio is a full-stack web application inspired by Airbnb that allows users to  
view, add, edit, and delete rental listings.  
This project focuses on backend development, RESTful APIs, and database integration.

---

## 🚀 Features
✅ View all rental listings  
➕ Add new rental properties 
✏️ Edit existing listings  
🗑️ Delete listings  
🔁 RESTful CRUD operations  
🎨 Clean and simple UI  

---

## 🛠️ Tech Stack
- 🌐 Frontend: HTML, CSS, EJS  
- ⚙️ Backend: Node.js, Express.js  
- 🗄️ Database: MongoDB  
- 🔧 Version Control: Git, GitHub  

---

## ▶️ How to Run Locally

### 1) Clone the repository
git clone https://github.com/VaibhavSingh2006/Rentio.git .

2️⃣ Go to the project folder
cd Rentio

3️⃣ Install dependencies
npm install

4️⃣ Start the server
node app.js

5️⃣ Open in browser 🌍
http://localhost:8080/listings
